This project focuses on predicting customer churn in the telecommunications industry using a Logistic Regression model. Customer churn—when users discontinue a service—is a critical business problem, as retaining existing customers is often more cost-effective than acquiring new ones .

Using the Telco Customer Churn dataset, this project performs end-to-end analysis, including data cleaning, exploratory data analysis (EDA), feature engineering, and model building. The goal is to identify key factors driving churn and build an interpretable model that estimates the probability of a customer leaving.

Through this analysis, the project not only demonstrates a practical machine learning workflow, but also highlights how data-driven insights can support customer retention strategies and improve business decision-making.
